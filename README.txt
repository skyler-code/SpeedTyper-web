SpeedTyper MVC Edition for my .NET 3 Final Project

This is a ASP.Net frontend for my project, SpeedTyper. Based in C#, the project uses a combination of SignalR and jQuery to provide interactivity for the client. Program is intended to display knowledge of ASP.Net, MVC architecture, dependency injection, web sockets, and Entity framework.

See it in action here: https://speedtyper.azurewebsites.net
